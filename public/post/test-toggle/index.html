JavaScript API

The toggle functionality provides a JavaScript API:

  1. window.toggleSection(sectionId) - Toggle a specific section by ID
  2. window.toggleAllSections('expand') - Expand all sections
  3. window.toggleAllSections('collapse') - Collapse all sections
  4. window.toggleAllSections() - Toggle all sections individually

Try running these in your browser console!

Usage in Markdown

To use toggle sections in your markdown files:












    

        
Section Title

        
    

    

        Your content here, with markdown support.

Parameters:

  • title: The header text (supports markdown)
  • collapsed: “true” to start collapsed, “false” (default) to start expanded
  • id: Optional unique ID for JavaScript access
  • class: Optional CSS class for styling (primary, success, warning, danger, info)
